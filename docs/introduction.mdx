## What is gitlog-weekly?

`gitlog-weekly` is a zero-dependency Node.js CLI tool that generates weekly git activity reports across one or multiple repositories.

Instead of manually running `git log` and piecing together what you or your team did last week, `gitlog-weekly` produces a clean, formatted summary in seconds.

```bash
npx gitlog-weekly --since "7 days ago" --repos ./api ./frontend ./shared
```

## Use Cases

- **Weekly standup prep** — see exactly what you committed last week before the meeting
- **Manager reports** — summarize team activity across all repos
- **Progress tracking** — add to your weekly review workflow
- **Changelog generation** — extract commits by type for release notes
- **CI reporting** — JSON output feeds into dashboards and notifications

## Quick Start

```bash
# Current directory, last 7 days
npx gitlog-weekly

# Multiple repos
npx gitlog-weekly --repos ./api ./frontend ./mobile

# Markdown output for copy-paste into reports
npx gitlog-weekly --format markdown

# JSON for automation
npx gitlog-weekly --format json | jq '.summary'
```
